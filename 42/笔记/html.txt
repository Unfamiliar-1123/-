1.<html> 与 </html> 之间的文本描述网页
  <body> 与 </body> 之间的文本是可见的页面内容
2.p标签：段落
3.h标签：标题  h1~h6（由大到小）
4.块级元素：
          有h,p,div,ul li,table tr td
          自己占一行，不与其他块级元素占同一行
5.行级元素：
          可与其他行级元素共处一行
          设置宽高无效
          但img可设置宽高
6.a  链接标签
7.input type="text"  输入框
8.img 图片链接
9.b strong  加粗
  em i 斜体
  br  空行
  del 删除线
10.表单元素：
           label for="**"  **与下方input中的name  id相同，才会在用户点击文字时 选中相应的输入框
           input:type的值
                 text 输入框
                 radio 单选框
                 checkbox 复选框
                 password  密码框
                 file  选择文件
                 email  邮箱框
                 url  网址框
                 number  数字框（有min max属性  最大最小值）
                 range  范围框（进度线）
                 date  日期框

                 button 按钮
                 reset 重置      这三条上面加button才会有按钮显示
                 submit 提交
            select  option：下拉框（多个选项）
            textarea（cols 列  rows 行）：可等比放大输入框
11.媒体元素
          video  插入视频
          autio  插入音乐
          header  头
          footer  尾
          内联框架：iframe src"所要引入的网页地址"
